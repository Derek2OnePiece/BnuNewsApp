1. 注册接口

appserver/user/register

请求方式：
POST

必选参数：
username
password
name
usertype

选填参数：
phone
sid - 学号或教师工号

返回数据：
{
"code":0,
"msg":"",
"userid":"",
"username":"",
"name":"",
"usertype":"",
}

示例：
{"username": "luojing.derek@gmail.com", "code": 0, "name": "derek", "userid": "53159347555eb23068a05321", "usertype": "1", "msg": "\u6ce8\u518c\u6210\u529f"}
